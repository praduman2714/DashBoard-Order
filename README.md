# Ajargh Kreation Assignment

Frontend Assignment for Ajargh Kreation intern role. 


## Installation

Clone the project
```bash
  https://github.com/Arinaymaybhaskar/AKassignment.git
```
Open project directory
```bash
  cd .\moobileapp\
```
Install dependencies
```bash
  npm install
```
Start the Application
```bash
  npm start
```
